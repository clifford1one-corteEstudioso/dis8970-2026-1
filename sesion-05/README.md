## Ambientes muy controlados

hablamos de demencia frontotemporal: desinocuación, deshinibición, control de impulsos reducido, inhibición reducida.

Esto lo mencionamos para hablar que los síntomas no están necesariamente asociados a la enfermedad. POdemos tener el 100% de los síntomas de una enfermedad o condición, y aún así no padecer aquella enfermedad.

### MoBi: mobile brain/body imaging

el limitante hasta ahora es que no se podía analizar la actividad cerebral mientras se está en funcionamiento.

es muy difícil también realizar procedimientos instrusivos a niñes, por lo que es casi imposible estudiarlos para obtener info valiosa.

La idea es hacer un dispositivo lo menos instrusivo posible para estudiar la mayor cantidad de sujetos posibles. Guaguas, awelitos, personas con discapacidad, pacientes, etc.
